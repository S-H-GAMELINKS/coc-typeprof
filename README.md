# coc-typeprof

typeprof lsp extension for coc.nvim

## Install

`:CocInstall coc-typeprof`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-typeprof-keymap)`

## Lists

`:CocList demo_list`
