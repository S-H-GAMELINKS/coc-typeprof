# coc-typeprof

typeprof lsp extension for coc.nvim

## Install
### vim-plug

```
Plug 'S-H-GAMELINKS/coc-typeprof'
```

## Keymaps

```
nmap <silent> <space>df <Plug>(coc-definition)
nmap <silent> <space>rf <Plug>(coc-references)
nmap <silent> <space>dt <Plug>(coc-type-definition)
```
